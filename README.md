# Student Micro-Learning Habit Recommendation System (Streamlit)

This is a simple AI-based web application that recommends **personalized micro-learning tasks**
to students based on their study behavior and preferences.

## How to run locally

1. Create a virtual environment (optional but recommended)
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Start the app:
   ```bash
   streamlit run app.py
   ```

The app will open in your browser at `http://localhost:8501`.

## Deploying

You can easily deploy this project on **Streamlit Community Cloud**:

- Push `app.py` and `requirements.txt` to a GitHub repository
- Create a new app on Streamlit Cloud and point it to `app.py`
